# 🚀 IntelliPrep: AI-Powered Placement Ecosystem

**IntelliPrep** is a full-stack web application designed for university students to track their placement readiness. It aggregates data from **LeetCode** and **GitHub** into a unified dashboard, providing a "Novelty" metric for SDE-1 readiness.

## ✨ Key Features

*   **📊 Dynamic Developer Dashboard:** Real-time synchronization with LeetCode and GitHub APIs to display problem-solving stats and contribution heatmaps.
*   **🛡️ Educational Firewall:** A moderated learning environment that filters distraction-free educational content for focused preparation.
*   **📈 Competency Mapping:** A technical radar chart that maps developer skills and identifies gaps in a student's profile.
*   **🔗 Public Profile Sharing:** Allows students to share their consolidated developer stats with recruiters via a unique public URL.

## 🛠️ Tech Stack

*   **Frontend:** Next.js (App Router), Tailwind CSS
*   **Backend:** Java, Supabase (Database & Auth)
*   **Integrations:** GitHub REST API, LeetCode GraphQL API
*   **Language:** TypeScript (99.2%)

  ## 🔒 Security & Version Control

*   **Environment Variables:** Sensitive API keys and database credentials are kept secure in `.env` files .
